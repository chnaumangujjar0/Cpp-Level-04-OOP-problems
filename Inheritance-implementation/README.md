# 🔗 Inheritance Implementation — C++ OOP

A collection of C++ programs demonstrating the three types of inheritance in Object-Oriented Programming.

## 📁 File Structure

| File | Description |
|------|-------------|
| `single_inheritance.cpp` | One base class, one derived class |
| `multilevel_inheritance.cpp` | Chain of classes — A → B → C |
| `multiple_inheritance.cpp` | One derived class from multiple base classes |

## Compile & Run
```bash
g++ -o single single_inheritance.cpp
g++ -o multilevel multilevel_inheritance.cpp
g++ -o multiple multiple_inheritance.cpp
```

## Concepts Covered
- Single Inheritance
- Multilevel Inheritance
- Multiple Inheritance
- `protected` access modifier
- Constructor chaining
