# 🎓 Student Class — C++ OOP Fundamentals

A beginner-friendly C++ project that demonstrates core **Object-Oriented Programming** concepts through a `Student` class with real-world attributes and behaviors.

---

## 📌 Concepts Covered

- **Encapsulation** — private data members with public methods
- **Parameterized Constructor** — with initializer list
- **Input Validation** — marks range enforced (0–100)
- **`const` Methods** — read-only functions that don't modify object state
- **Conditional Logic** — login/logout state management

## ▶️ Getting Started

### Prerequisites
- A C++ compiler: `g++`, `clang++`, or MSVC
- C++11 or later

### Compile & Run

```bash
# Compile
g++ -o student student_class.cpp

# Run
./student
```
