# 🚗 Car Class — C++ OOP

A simple C++ program that simulates basic car operations using Object-Oriented Programming.

## Concepts Covered
- Encapsulation
- Constructor with input validation
- Destructor
- `const` methods

## Compile & Run
```bash
g++ -o car car_class.cpp
./car
```

## Sample Output
```
Reached maximum speed!
Model: Toyota
Brand: Corolla
Speed: 180
Max Speed: 180

Model: Toyota
Brand: Corolla
Speed: 0
Max Speed: 180

Car object destroyed
```

## Features
- Accelerate with max speed cap
- Brake with minimum speed of 0
- Input validation in constructor
- Destructor to track object lifecycle
