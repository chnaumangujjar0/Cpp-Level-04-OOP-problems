# 🏦 Bank Class — C++ OOP

A simple C++ program that simulates basic banking operations using Object-Oriented Programming.

## Concepts Covered
- Encapsulation
- Constructors with initializer list
- Input validation
- `const` methods

## Compile & Run
```bash
g++ -o bank bank_class.cpp
./bank
```

## Features
- Login / Logout system
- Deposit with validation
- Withdraw with balance check
- View account balance (only when logged in)
